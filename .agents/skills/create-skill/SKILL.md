# 目标

生成简洁、可复用的 skill。

## 工作流

1. 明确用途、输入、输出
2. 生成英文 name（小写+连字符）
3. 写中英 description（能力+触发条件）
4. 写中文正文（规则为主）

## 规则

- 正文用中文
- description 必须中英双语
- 不要冗长：只保留必要规则
- 不要写背景说明或解释
- 非必要不创建额外目录

## 输出

- skill 目录
- SKILL.md

## 自检

- name 是否规范
- description 是否清晰
- 是否可以一屏读完
